# Security Policy

## Reporting a Vulnerability

Please do NOT open public issues for security vulnerabilities.

Report privately via the GitHub security advisory feature or contact the maintainer directly.

## Known Limitations

- `BaseLazyMint` relies on ECDSA signature verification — protect your signing key carefully
- `BaseNFTMarket` does not have escrow — NFTs must be approved before listing
- `BaseNFTStaking` reward token supply must be manually funded
- These contracts have NOT been professionally audited

## Disclaimer

Educational and experimental use only. Use at your own risk.
